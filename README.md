You can see our preprint on arXiv: https://arxiv.org/abs/2604.19499


## Overview

This repository provides a reproducible implementation of a unified framework for interpreting and extending Delta-based stylometric distance measures.

The project introduces:

- probabilistic extensions of Delta (Jensen–Shannon Delta)  
- rank-based extensions (Rank-Turbulence Delta)  
- token-level decompositions of stylistic distance  
- robustness analyses of interpretability

The repository supports both exploratory stylometry and attribution experiments.

### Notebooks

- `pam_multilingual.ipynb`  
Multilingual clustering experiments (EN/DE/FR) using PAM and evaluation of Rank-Turbulence parameter α.
- `plot_comparison_figures.ipynb`  
Reproduces all main figures: clustering curves, α sensitivity plots, attribution results, UMAP projections, and lexical contribution visualisations.
- `russian_tf.ipynb`  
Preprocessing pipeline for Russian texts (cleaning, lemmatisation, TF construction, optional n-grams).
- `token_impact_validation.ipynb`  
Robustness experiments:
  - stability under MFW variation  
  - bootstrap resampling  
  - token removal validation


## Data

### Multilingual corpora (EN / DE / FR)

The multilingual datasets are derived from the public benchmark repository:

[https://github.com/schtepf/ExplainingDelta](https://github.com/schtepf/ExplainingDelta)  

These data are reused for comparability with previous studies.

Expected files:

- `data/TfMatrixEN.json`, `data/authorsEN.json`  
- `data/TfMatrixDE.json`, `data/authorsDE.json`  
- `data/TfMatrixFR.json`, `data/authorsFR.json`


### Russian corpus (SOCIOLIT)

Russian texts are sourced from the SOCIOLIT corpus:

[https://sociolit.ru](https://sociolit.ru)  

Due to copyright restrictions, full texts are **not redistributed** in this repository.

Instead, we provide:

- corpus notes: `data/Russian/README.txt`  
- a small **public-domain** sample for optional re-preprocessing: `data/Russian_sample/` (`.txt` files)  
- precomputed term–frequency inputs used in the paper: `data/TfMatrixRU.json`, `data/authorsRU.json` (and optional `data/TfMatrixRU_3-5gram.json`)

This ensures transparency while respecting licensing constraints.


## Preprocessing

The preprocessing pipeline follows the methodology described in the article:

For Russian texts:

1. Retain Cyrillic and Latin alphabetic characters only
2. Convert to lowercase
3. Apply lemmatisation using `pymorphy3`
4. Do not remove stopwords
5. Construct TF matrices from the top 20,000 types
6. Use the first 10,000 most frequent words (MFW) for experiments

For Jensen–Shannon Delta:

- apply additive smoothing prior to normalisation


## Implemented Methods

- Burrows’s Delta (L1 distance on z-scores)  
- Quadratic (Euclidean) Delta (L2)  
- Cosine Delta  
- Jensen–Shannon Delta  
- Rank-Turbulence Delta (parameterised by α)

All methods support **token-level contribution decomposition**, enabling interpretability.


## Reproducibility

### Environment

Python ≥ 3.10 is recommended.

Core dependencies:

- numpy  
- pandas  
- scipy  
- scikit-learn  
- scikit-learn-extra  
- umap-learn  
- matplotlib  
- seaborn  
- tqdm  
- pymorphy3

Install:

```bash
pip install -r requirements.txt
```

To run the `.ipynb` files interactively, use JupyterLab or Notebook (`pip install jupyter`), or open them in an editor that integrates with your Python environment (for example, VS Code / Cursor with the Jupyter extension).


## Outputs

The notebooks reproduce all experimental results reported in the paper, including:

- clustering performance curves (ARI)  
- authorship attribution (balanced accuracy)  
- UMAP visualisations  
- lexical contribution plots  
- robustness diagnostics (Jaccard overlap, bootstrap stability)
